public sealed class SerializerBuilder

assembly YamlDotNet
namespace YamlDotNet.Serialization
Base type:  BuilderSkeleton<SerializerBuilder>
Creates and configures instances of Serializer. This class is used to customize the behavior of Serializer. Use the relevant methods to apply customizations, then call SerializerBuilder.Build to create an instance of the serializer with the desired customizations.
Attributes
[RequiresDynamicCode("This builder configures the serializer to use reflection which is not compatible with ahead-of-time compilation or assembly trimming. You need to use the code generator/analyzer to generate static code and use the 'StaticSerializerBuilder' object instead of this one.")]
