public sealed class DeserializerBuilder

assembly YamlDotNet
namespace YamlDotNet.Serialization
Base type:  BuilderSkeleton<DeserializerBuilder>
Creates and configures instances of Deserializer. This class is used to customize the behavior of Deserializer. Use the relevant methods to apply customizations, then call DeserializerBuilder.Build to create an instance of the deserializer with the desired customizations.
Attributes
[RequiresDynamicCode("This builder configures the deserializer to use reflection which is not compatible with ahead-of-time compilation or assembly trimming. You need to use the code generator/analyzer to generate static code and use the 'StaticDeserializerBuilder' object instead of this one.")]

Registers a TypeDiscriminatingNodeDeserializer to be used by the deserializer. This internally registers all existing INodeDeserializer as inner deserializers available to the TypeDiscriminatingNodeDeserializer. Usually you will want to call this after any other changes to the INodeDeserializers used by the deserializer.
